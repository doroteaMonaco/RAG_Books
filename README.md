# RAG Books Exercises

This repository contains all the exercises and code implementations from the following three books regarding Retrieval-Augmented Generation (RAG):

- **RAG Driven Generative AI**
- **Building AI Agents with LLMs, RAG, Knowledge Graphs**
- **Hands-On RAG for Production**

## Repository Structure
The workspace is organized into folders corresponding to each book. Each folder contains the exercises, Jupyter notebooks, and related code for that specific book.
